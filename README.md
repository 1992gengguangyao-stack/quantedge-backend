## Current product scope

- Wallet-only SIWE authentication, including WalletConnect-compatible mobile wallets
- Saved strategy and daily backtest limits enforced by plan
- Fee- and slippage-aware historical research using OHLCV data
- Hyperliquid testnet preparation with signatures produced locally in the browser
- USDT payments on TRC-20 with exact-amount matching and on-chain confirmation
- Privacy-first product analytics and an SEO/GEO-ready public site

BTCquant is research software. It does not promise returns and does not take custody of exchange or wallet private keys. Users should evaluate the Free plan before purchasing a subscription or using any strategy with real capital.
